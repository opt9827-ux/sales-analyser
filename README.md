# Sales Analyzer

A Python + SQL project that analyzes superstore sales data.

## Tech Stack
- Python
- Pandas
- SQLite
- SQL

## Features
- Sales analysis by region, category, segment
- Top and bottom product performance
- Monthly profit trends
- Customer and shipping analysis
- Interactive menu driven interface

## How to Run
1. Clone the repo
2. Install dependencies: pip install pandas
3. Add superstore.csv to the project folder
4. Run: python main.py

## Skills Demonstrated
- Advanced SQL queries (GROUP BY, HAVING, subqueries, date functions)
- Python data handling with Pandas
- SQLite database integration
- Clean modular code structure
